# LHCD
The LHCD dataset is a time-series dataset for cropland non-agriculturalization, consisting of 1295 samples that cover 23 image periods. Each sample has a spatial extent of 512 × 512 pixels. The dataset encompasses major non-agriculturalization types such as newly constructed buildings, landscaped areas, roads, bare land, and ponds. Examples of the samples are presented in Fig. 1.
